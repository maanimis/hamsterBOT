# hamsterBOT
simple autiTap bot for https://t.me/hamster_kombat_bot

## run@linux
```
git clone git@github.com:maanimis/hamsterBOT.git 
cd hamsterBOT
```
edit 'Authorization' header from index.js and save it:
```
nano index.js
```
install dependencies:
```
npm i
```

run:
```
npm start
```
